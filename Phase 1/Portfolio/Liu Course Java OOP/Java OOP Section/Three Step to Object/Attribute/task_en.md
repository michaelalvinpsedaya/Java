## Attribute/Attr properties

- Properties identify the characteristic information of a class or object.
- What was previously called "variables" in Java are usually referred to as "properties" in object-oriented programming. Not all variables are properties, but there's no need to deliberately distinguish them.

<br>

## Attribute/Attr properties vs Variable

- Properties have certain real-world significance, such as the id, name, and age of a `Person` class. Variables have more programmatic significance, like the i, j in a `for` loop.
- Properties are more focused on describing the characteristics of a real-world entity, whereas variables might only be used in a small scope and are the "amount" that can change.

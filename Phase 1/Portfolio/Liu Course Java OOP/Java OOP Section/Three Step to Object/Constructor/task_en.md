# What are the steps to create a person?

---

<br>

## Constructor methods

- The process of creating **an object of the class** based on a `class` is called **construction**
- The method used to create an object is called a **constructor method**
- The name of the constructor method is the same as the class name, such as the constructor method `Person()` for `class Person`
- All classes have their default constructor method, which you can explicitly define and modify
- When defining a constructor method, it is "**no return (but not `void`)**", doesn't declare a return type, and cannot use `return`, because the "return" type of a constructor method is the class itself
- A constructor method represents the actions to perform when creating this object; otherwise, a "bare" object is created

<br>

## Creating an object with new

The constructor method is a special type of method that can only be called using the `new` keyword.
